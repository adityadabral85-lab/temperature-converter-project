Temperature Converter Website

Convert temperatures between Celsius, Fahrenheit, and Kelvin.

Enter a temperature, choose the original unit, choose the unit you want, and press convert. The result will appear with the correct unit.

What this project includes

  • Input validation for temperature values
  • Dropdown menus for selecting units
  • A convert button that triggers the calculation
  • A result area that displays the converted value and unit

Conversion formulas

  • Celsius to Fahrenheit: C x 9/5 + 32
  • Fahrenheit to Celsius: (F - 32) x 5/9
  • Celsius to Kelvin: C + 273.15
