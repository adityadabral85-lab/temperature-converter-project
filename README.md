# Temperature Converter Website

A beginner-friendly temperature converter website made with HTML, CSS, and JavaScript.

## Features

- Temperature input field
- Dropdown menus for selecting units
- Convert button
- Result display area
- Input validation
- Celsius, Fahrenheit, and Kelvin support
- Responsive layout for mobile and desktop

## Files

- `index.html` - Page structure and converter form
- `style.css` - Website styling and responsive layout
- `script.js` - Temperature conversion logic and validation

## How to Run

Open `index.html` in your browser.

You can also use the VS Code Live Server extension while editing.

## How to Push to GitHub

```bash
git init
git add .
git commit -m "Add temperature converter website"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPOSITORY.git

```

Replace `YOUR-USERNAME` and `YOUR-REPOSITORY` with your GitHub account and repository name.
